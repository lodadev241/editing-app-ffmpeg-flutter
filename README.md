## What is FFmpeg?

## Basic options
1. Global options:
- **-y**: Overwrite the output
2. Input options:
- **-i**: The input file
3. Output options:
- **-c:v**: version of codec:c

## Commands
1. Trim
    - Use the "-ss" flag. The value can be specified in s/ms or as a timestamp
    - Options:
        - -t: This sets the duration
        - -to: this sets the timestamp where the input video should stop
